# GR4553-Group1
Group 1 Final Project Repository 
Contributors: Taylor Hunn, Ashton Champion, Dawson Smith
This repository contains plots and code covering the severe weather event in the Texas Panhandle on April 25,2025. 
CODE: All code for this project is located in the folder labelled "CODE". We have separated the code based upon who created and wrote said code. Contained in "teh287 CODE" is the code for the 2 meter high and low temperatures, 300mb geopotential heights and wind barb map, 700mb geopotential map, morning and evening soundings from KAMA, dewpoints from the morning and evening showing the advancement of the dryline, low level shear, full column shear, surface based CAPE, surface based CIN, and the reports map. 
The plots generated by our codes can be found within the "IMAGES" folder under the resepective creator. 
DATA:
CSV files were taken from the SPC event archive page to create the reports map. We also included a link to where we accessed our GRBS data.
